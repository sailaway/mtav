# [MTAV - 多轨道动画视频生成器][mtav-git]

> MTAV 是一个多轨道视频内容合成工具，每条轨道在不同的时间点根据对应的属性生成视频画面，通过定义属性随着时间的变化过程生成动态视频内容。最终所有轨道内容组合生成最终的视频画面。
